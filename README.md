## Overview

David-stack is a modern developer portfolio website designed to showcase projects, skills, experience, and coding activity through a visually aggressive and highly interactive interface.

The website combines brutalist layouts, terminal-inspired visuals, smooth animations, glitch effects, and modern frontend engineering into a fully responsive experience optimized for desktop and mobile devices.

---

# Features

- Brutalist cyberpunk design system
- Interactive boot screen
- Custom animated cursor
- Smooth scrolling experience
- Glitch hover animations
- Fully responsive layout
- Animated marquee banners
- Dynamic coding stats section
- GitHub integration
- Modern project showcase
- Advanced UI transitions
- Performance-focused structure
- Neo-brutalist shadows and typography
- Terminal-inspired interface elements

# Installation

Clone the repository:

```bash
git clone https://github.com/david1337x/david-stack.git
```

Move into the project directory:

```bash
cd david-stack
```

Open the project:

```bash
Open index.html in your browser
```

Or run with Live Server inside VS Code.
